# CSC 1103 Tic-Tac-Toe Game

# Build
Build with 
```
gcc -I src/SDL2 -L src/lib -g logic.c render.c -o main main.c -lmingw32 -lSDL2main -lSDL2 -lSDL2_image -lSDL2_ttf
``` 


# Run program 
To run the Tic Tac Toe game 
```
./main.exe
```
